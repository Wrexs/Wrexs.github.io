Angry Birds
An early project during my first year at UWE was to replicate the game Angry Birds.

This implementation includes a self-made physics system for the birds along with different powerups for each bird, such as piercing blocks.

Download

Space Rings - EGA Trek Clone
The goal of this project was to replicate the style of EGATrek, a DOS game themed around sci-fi classics.

This implementation includes a dynamic map and a state-based gameplay loop.

Download

Physics Based Marrionette System
This project involved creating a physics-based controller for a marrionette puppet within Unity and Unreal.

From this project I learned how to apply UI elements to effects that I create, so that other users / developers can easilly adjust the effect to fit their needs.

Lovecraftian Horror Platformer
For this project I was tasked with creating a short (4 minute) playable sequence within Unreal Engine.

From creating this level I furthered my knowledge of the tools within Unreal Engine, Blueprints / Scripts and iterating design to create challenges / goals.
